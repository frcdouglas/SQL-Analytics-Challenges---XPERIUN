Este repositório foi criado com o objetivo de documentar minha evolução prática em SQL por meio da resolução de desafios voltados para análise de dados, lógica de negócio e interpretação de informações.

A proposta aqui não é apenas resolver exercícios, mas também demonstrar meu raciocínio analítico, a forma como estruturo consultas e como penso em diferentes maneiras de chegar a uma mesma resposta.

## Objetivo do repositório

Este portfólio foi desenvolvido para reunir estudos e resoluções de desafios em SQL, com foco em fortalecer habilidades importantes para áreas como:

- análise de dados
- business intelligence
- analytics engineering
- engenharia de dados (base analítica)

Ao longo dos desafios, meu objetivo é mostrar não apenas o resultado final, mas também o processo de pensamento por trás de cada solução.

## O que você vai encontrar aqui

Cada desafio foi organizado em sua própria pasta e pode conter:

- descrição do problema
- resolução em SQL
- explicação do raciocínio utilizado
- comentários sobre por que determinada abordagem foi escolhida
- comparações entre diferentes formas de resolver a mesma questão
- possíveis melhorias ou alternativas de escrita
- insights de negócio quando aplicável

## Como este repositório foi pensado

Minha intenção com este material é ir além da execução técnica e construir um espaço que mostre:

- capacidade de interpretar problemas de negócio
- organização e clareza na escrita de queries
- entendimento sobre boas práticas em SQL
- comparação entre soluções diferentes
- preocupação com documentação e comunicação técnica

Ou seja, este repositório funciona tanto como prática quanto como portfólio profissional.

## Como navegar pelo conteúdo

A melhor forma de explorar este repositório é:

1. acessar a pasta do desafio desejado  
2. ler o contexto e a proposta no `README.md` daquele desafio  
3. visualizar a resolução no arquivo `queries.sql`  
4. acompanhar as explicações e comparações nos arquivos complementares  
5. consultar imagens ou materiais extras, quando houver
